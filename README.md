# EVTEG (TEG Web App)

Sistema de evaluación y gestión del Trabajo Especial de Grado (TEG).  
Permite el seguimiento de entregas, la colaboración entre estudiantes y tutores, y la evaluación por jurados.

## Características
- Entregas y plazos: registro de entregas y vistas de fechas límite.
- Colaboración: intercambio de archivos y comunicación entre equipo y tutor.
- Recursos y documentación: guías, plantillas y material de apoyo.
- Evaluación: interfaz para revisión y calificación de entregas.
